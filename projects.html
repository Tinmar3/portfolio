Most of my projects consist of classic HTML+CSS+JS structure...

...but some of them needed extra piece of magic.

Salvus health Big medicines selling company. Goal was to create friendly and warm’ web page from design, with lot of animations. Page has custom advanced scrolling by sections in home page, advanced menu in about us and is gorgeous all around. In the end it’s made Vanilla, with the help of Webpack and some smaller libraries like zenscroll. SCSS handled CSS mess. Best web page I've made till this day, design complexity and code cleanliness wise.

DuAdvisor Like its name says, this app serves for advising and informing purposes for the City of Dubrovnik. I created slice from design with the help of JQuery. It has a lot of subpages and many different components, like any other advanced application. Later, back-end dev of this app implemented AngularJS and CMS.

JBC The oil and gas market's research center. They needed an application for handling big amount of data of their reports, news, company informations and so on, for every user of the app. The best solution was to implement ReactJS – there are a lot of components and subpages so dynamical rendering just made sense. SCSS also helped here.

Smart-sense Smart sensors development company. Their web page is really simple, except it has custom scrolling by full page sections – and that’s why I've used fullPage.js JQuery library. One of the first web pages I’ve ever made from scratch.

Smart-net IT networks and technology implementation company. Really simple page with nice middle menu for navigation through their areas of work. Page didn’t require any special technologies, SCSS was only used for better CSS organization.

Magdalena polyclinic Special hospital for cardiovascular surgery and cardiology. Lot of data, lot of text and lot of information needed to be shown on the page. It was built with JQuery for easier making of cool sliders and other data visualization containers. One of the first web pages I've made from scratch, too.
