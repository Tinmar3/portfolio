Salvus health *

Big medicine selling company. Goal was to create friendly and warm web page from design, with lot of animations. Page has custom advanced scrolling by sections in home page, advanced menu in about us page and is gorgeous all around. In the end it’s made Vanilla (JS), with the help of Webpack and some smaller libraries like zenscroll. SCSS handled CSS's mess. Best web page I've made 'till this day, design complexity and code cleanliness wise.
